# auto-clicker-python
Projeto de automação de eventos em Python

Requisitos*
Python instalado.
VS Code instalado.

Como usar*
1-Abra o projeto no VS Code.
2-Execute o arquivo principal em modo Debug.
3-Após iniciar, pressione F2 para ativar o script.
4-O auto clicker começará a clicar automaticamente, sem necessidade de usar o mouse.

Configurar*
Na linha 25 do código, você pode alterar o tempo de intervalo entre os cliques.
O valor deve ser definido em milissegundos.
